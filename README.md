# 🌸 Bloom Filter Visualizer

This project visualizes the working of a **Bloom Filter**, a probabilistic data structure used for fast membership checking.

## 📌 Features

* Interactive frontend visualization
* Backend logic for Bloom Filter operations
* Demonstrates hashing and false positives
* Educational tool for understanding data structures

## 🧠 What is a Bloom Filter?

A Bloom Filter is a space-efficient probabilistic data structure used to test whether an element is present in a set.

* No false negatives
* Possible false positives

## 📂 Project Structure

* `frontend/` → UI and visualization
* `backend/` → Bloom Filter logic
